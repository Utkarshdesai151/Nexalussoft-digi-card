# Nexalussoft Business Card

This project is a digital business card for Nexalussoft, showcasing the company details, contact information, and services. The business card features sections such as company profile, contact details (Phone, WhatsApp, Email), and more. It also includes a form to submit inquiries, and a QR code for easy sharing.

## Features

- **Company Details**: Displays information about the founder and CEO, Ravi Senjaliya.
- **Contact Methods**: Phone, WhatsApp, and Email links for easy communication.
- **Download VCard**: Option to download the vCard for adding the contact to the phone.
- **Inquiry Form**: A modal to send inquiries via form submission or SMS.
- **Web Development Services**: Displays technology stack (Python, Django, React, etc.) used for web development.
- **Accordion for Services**: Includes an accordion for displaying contact information, services, and mobile app development details.

## Tech Stack

- **Frontend**: 
  - HTML5
  - CSS (Tailwind CSS for styling)
  - Font Awesome (for icons)
  - JavaScript (for interactive modal and accordion)

- **Other Tools**:
  - **QR Code Generation**: Uses the QR Code Library to generate QR codes for sharing.
